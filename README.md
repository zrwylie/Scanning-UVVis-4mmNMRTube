# Scanning-UVVis-4mmNMRTube
Hardware and software to build and adapt a syringe to take UV-Vis spectra scans along a standard 4mm OD NMR tube.

Scanner adapted from syringe: [Repo Name](https://github.com/pachterlab/poseidon.git)
Interfacing with OceanInsight Spectrometers: Python-SeaBreeze (https://python-seabreeze.readthedocs.io/en/latest/#)
